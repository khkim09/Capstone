Detailed Description

이벤트 트리 맵을 생성하고 시각화하는 매니저. 노드를 자동으로 배치하고, 위험 여부를 표시하며, 연결선까지 구성합니다. 노드 클릭 시 선택 이벤트를 발생시킵니다.
